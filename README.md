# ♟ Chess Project

A desktop Chess application developed in **JavaFX** with **MySQL** support and **Stockfish 17.1 AI**.

## Features

- Full chess gameplay
- Player vs Player mode
- Player vs AI mode using Stockfish 17.1
- 20 AI difficulty levels
- User login and registration
- Game history saved to MySQL
- Settings page
- Enable or disable premoves
- Fullscreen mode
- Board color selection
- Sound effects
- Resign option
- Play Again option
- Return to Main Menu after game ends

## Technologies Used

- Java 21
- JavaFX
- Maven
- MySQL
- JDBC
- FXML
- Stockfish 17.1
